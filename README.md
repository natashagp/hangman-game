# Hangman Game

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/hangman-game.git
```

### Step 2

Change the directory to hangman-game

```shell
cd hangman-game
```

### Step 3

Run project

```shell
python main.py
```
